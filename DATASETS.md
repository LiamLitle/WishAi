## 🧭 How to choose?

| Goal | Recommended datasets |
|------|---------------------|
| **General French model** | Wikipedia FR + mC4 + CulturaX + Gutenberg FR |
| **Smart assistant** | OpenHermes 2.5 + UltraChat + SlimOrca |
| **Code copilot** | The Stack v2 + CodeContests + APPS + HumanEval |
| **Math model** | NuminaMath + MetaMath + GSM8K + MATH |
| **Medical** | PubMed + MedQA + Clinical Notes |
| **Multilingual** | mC4 Multi + CC-100 + OPUS-100 + Wikipedia (all languages) |
| **Legal** | Pile of Law + French Legal Texts |
| **All-in-one** | RedPajama or Dolmino (complete pretraining corpora) |
